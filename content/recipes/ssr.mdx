# 서버 사이드 렌더링

서버 사이드 렌더링를 사용하면 컴포넌트들은 한 번만 렌더링 됩니다. 컴포넌트는 unmounted 되지 않으므로 이 경우 observer 컴포넌트는 서버 측에서 렌더링 될 때 메모리를 누출합니다.

메모리 누수를 피하려면 서버 사이드 렌더링을 사용할 때 `useStaticRendering(true)`를 호출하여 observer를 본질적으로 비활성화하세요.

```ts
import { useStaticRendering } from 'mobx-react'

// 원하는 조건을 사용하세요. 이것은 단지 예시입니다.
if (process.env.SSR) {
  useStaticRendering(true)
}
```

> 참고: 명명에도 불구하고, 이 함수는 리액트 훅이 아닙니다. 이름은 리액트 훅이 소개되기 오래 전에 결정되었습니다. 서버측 코드의 루트에서 한번만 호출해야 합니다.
